# Starter Template for Gulp, Jade, and SASS




## Project Folder Structure

```
dist/           -- main production folder
  assets/       -- all assets (css, fonts, imgs, js)
    css/
    fonts/
    imgs/
    js/
  index.html
src/            -- all source files
  jade/
    includes/
      primary-navigation.jade
      secondary-navigation.jade
    layouts/    -- jade page layouts
      index.jade
      page.jade
    index.jade
  scripts/
    vendor/
      jquery.js
    main.js
    navigation.js
  scss/
    addons/
    elements/
    setup/
    utilities/
    style.scss
gulpfile.js
package.json
```
